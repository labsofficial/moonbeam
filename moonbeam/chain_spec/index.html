get_account_id_from_pair

Helper function to get an AccountId from Key Pair We need the full decompressed public key to derive an ethereum-style account Substrate does not provide a method to obtain the full decompressed public key Therefore, this function uses the secp256k1_ecdsa_recover method to recover the full key A solution without using the private key would imply solving the secp256k1 curve equation The latter is currently not possible with current substrate methods
